# Streamlitのデプロイ

## デプロイ手順

1. https://share.streamlit.ioへ移動し、githubアカウントまたはgmailアカウントでサインイン
2. streamlit.ioのトップページへ移動したら、「New App」ボタンをクリック
3. リポジトリ、ブランチ、ファイルを選択する
4. 「deploy」ボタンをクリック
5. streamlitアプリが表示されれば成功

※デプロイしたアプリは今後、git pushによってすぐにアプリが更新されるようになる。
